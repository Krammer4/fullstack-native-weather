# Fullstack React Native + MongoDB + Express + NodeJS Weather forecast mobile app

СТЕК ТЕХНОЛОГИЙ:

- React Native
- Expo
- MongoDB
- Express
- NodeJS
- React Router
- Mongoose
- Weather API
- Maps
- bcryptjs
- cors
- jwt
- express-validator
- config
- nodemon

# To launch app:

- npm install (in frontend dir)
- npm install (in backend dir)
- set your params in backend/config
- npx expo start
